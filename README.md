# 📊 Data Analytics Project

## 🔹 Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL Server, and Power BI.
The goal is to extract meaningful insights from raw data by performing data cleaning, analysis, and visualization, and presenting results through an interactive dashboard and report.

---

## 📂 Dataset

* The dataset contains structured data used for analysis.
* It includes features relevant to business insights such as trends, patterns, and performance metrics.
* Data was initially loaded and processed using Python.

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL Server** (Data querying & transformation)
* **Power BI** (Dashboard creation & visualization)
* **Gamma** (Presentation creation)
* **GitHub** (Version control & project showcase)

---

## 🔄 Project Workflow

### 1. Data Loading

* Imported dataset using Python (Pandas)
* Explored structure, data types, and missing values

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized column formats
* Performed basic preprocessing

### 3. Exploratory Data Analysis (EDA)

* Univariate and bivariate analysis
* Identified patterns and trends
* Used visualizations for better understanding

### 4. SQL Analysis

* Loaded cleaned data into SQL Server
* Wrote queries for:

  * Aggregations (SUM, COUNT, AVG)
  * Filtering and grouping
  * Business-level insights

### 5. Power BI Dashboard

* Built interactive dashboard
* Visualized key metrics and KPIs
* Used charts, filters, and slicers for insights

### 6. Reporting & Presentation

* Created a structured report
* Designed a presentation using Gamma
* Highlighted key findings and insights
